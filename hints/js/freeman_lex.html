Ok, what should I start from?

You need string.replace()... 

mainText.replace(target, repl)

but this expression replaces only the first occurrence.

How to replace all substrings?

You need to create a new regular expression based on target and global modifier for search and replace all over the text. 

new RegExp(target, 'g')
What to do next?

The method replace() returns a new string, which you may immediately return out of the function.

So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.replace(new RegExp(target, 'g'), replacement);
}

Split the given text by target substring (all occurrences removed) and join parts back with a new string. 

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.split(target).join(replacement);
