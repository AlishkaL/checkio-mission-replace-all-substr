Ok, what should I start from?

You need str.replace() without the third argument since you want to replace all occurrences. 

mainText.replace(target, repl)
What to do next?

This method returns a new string, which you may immediately return out of the function.

So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def replace_all(mainText: str, target: str, repl: str) -> str:

    return mainText.replace(target, repl)
