出个主意💡！这个任务其实很简单。使用单字符串方法将目标子字符串替换为文本中的所有新字符串。

好吧，我应该从什么开始呢？

您需要string.replace()... 

mainText.replace(target, repl)

但该表达式只替换第一次出现的内容。

如何替换所有子串？

您需要在target 和全局修改器的基础上创建一个新的regular expression ，用于搜索和替换所有文本。 

new RegExp(target, 'g')
下一步该怎么做？

replace() 方法返回一个新字符串，您可以立即将其从函数中返回。

那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.replace(new RegExp(target, 'g'), replacement);
}

按目标子串分割给定文本（删除所有出现的子串），然后用新字符串连接各部分。 

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.split(target).join(replacement);
}
