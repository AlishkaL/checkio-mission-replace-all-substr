您需要不带第三个参数的str.replace() ，因为您想替换所有出现的内容。 

mainText.replace(target, repl)
下一步该怎么做？

此方法返回一个新字符串，您可以立即将其从函数中返回。

那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def replace_all(mainText: str, target: str, repl: str) -> str:

    return mainText.replace(target, repl)

按目标子串分割给定文本（删除所有出现的子串），然后用新字符串连接各部分。 

def replace_all(mainText: str, target: str, repl: str) -> str:

    return repl.join(mainText.split(target))
