Lassen Sie sich etwas einfallen 💡! Diese Aufgabe ist eigentlich sehr einfach. Verwenden Sie eine einzelne String-Methode, um eine Zielteilzeichenkette im gesamten Text durch eine neue zu ersetzen.

Ok, womit soll ich anfangen?

Sie benötigen str.replace() ohne das dritte Argument, da Sie alle Vorkommen ersetzen möchten. 

mainText.replace(target, repl)
Was tun Sie als nächstes?

Diese Methode gibt einen neuen String zurück, den Sie sofort aus der Funktion heraus zurückgeben können.

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def replace_all(mainText: str, target: str, repl: str) -> str:

    return mainText.replace(target, repl)

Klar, schnapp sie dir!

Teilen Sie den gegebenen Text nach der Zielteilzeichenkette auf (alle Vorkommen werden entfernt) und fügen Sie die Teile wieder zu einer neuen Zeichenkette zusammen. 

def replace_all(mainText: str, target: str, repl: str) -> str:

    return repl.join(mainText.split(target))
