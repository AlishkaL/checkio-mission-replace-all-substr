Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać jakiś pomysł 💡! Ta misja jest w rzeczywistości bardzo prosta. Proszę użyć metody pojedynczego ciągu, aby zastąpić docelowy podciąg nowym w całym tekście.

Ok, od czego powinienem zacząć?

Potrzebuje Pan string.replace()... 

mainText.replace(target, repl)

ale to wyrażenie zastępuje tylko pierwsze wystąpienie.

Jak zastąpić wszystkie podciągi?

Proszę utworzyć nowy regular expression na podstawie target i globalnego modyfikatora do wyszukiwania i zastępowania całego tekstu. 

new RegExp(target, 'g')
Co robić dalej?

Metoda replace() zwraca nowy ciąg znaków, który można natychmiast zwrócić z funkcji.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.replace(new RegExp(target, 'g'), replacement);
}

Jasne, proszę pobrać!

Proszę podzielić podany tekst przez docelowy podciąg (usuwając wszystkie wystąpienia) i połączyć części z powrotem z nowym ciągiem. 

function replaceAll(mainText: string, target: string, replacement: string): string {

    return mainText.split(target).join(replacement);
}
